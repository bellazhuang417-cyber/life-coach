# Life Coach Skill for WorkBuddy

一个人生教练 Skill，基于苏格拉底式追问 + 温暖直接的教练风格。

## ✨ 功能特点

- **苏格拉底式追问**：不直接给答案，而是通过提问帮你看见自己的盲区和模式
- **温暖但直接**：不做讨好型 AI，不替你做决定，但会轻轻戳破你的借口
- **洞察报告生成**：对话结束后可生成 HTML 格式的洞察报告（Kami 风格）
- **节奏控制**：自动检测疲劳信号，避免疲劳轰炸
- **可选个人上下文**：支持读取 Obsidian/Markdown 笔记系统建立更好理解（需用户授权）

## 🎯 适用场景

- 想不通某个决定，需要有人帮你理清楚
- 情绪困扰，想找人聊聊但不知道从哪开始
- 想做自我反思，但缺少一个"对话伙伴"
- 做完 Weekly Review，想深入聊聊某个话题

## 🚀 安装方法

### 方法一：从 GitHub 手动安装（推荐）

1. 下载本仓库：`git clone https://github.com/bellazhuang417-cyber/life-coach.git`
2. 将 `SKILL.md` 复制到你的 WorkBuddy skills 目录：
   ```bash
   cp life-coach/SKILL.md ~/.workbuddy/skills/life-coach/SKILL.md
   ```
3. 重启 WorkBuddy，skill 会自动加载

### 方法二：通过 WorkBuddy 命令安装

在 WorkBuddy 对话中输入：
```
/install-skill https://github.com/bellazhuang417-cyber/life-coach
```

## 💬 使用方法

安装后，在 WorkBuddy 对话中输入以下触发词之一：

- `人生教练`
- `life coach`
- `我想聊聊`
- `帮我理一理`
- `教练对话`
- `心理探索`
- `自我反思`
- `情绪梳理`
- `我想找你聊聊`
- `coach me`
- `help me reflect`
- `I need clarity`
- `给我一些建议`
- `我想不通`
- `很纠结`
- `不知道该怎么办`

## 📋 对话结构

1. **开场**：教练会根据你的描述选择合适的开场方式
2. **苏格拉底追问**：通过提问帮你看见问题本质（每 8-10 轮会主动暂停）
3. **节奏控制**：如果你显出疲惫，教练会主动收尾或切换节奏
4. **洞察报告**（可选）：对话结束后可生成 HTML 报告，包含：
   - Session 主题
   - 教练观察
   - 你的输出/领悟
   - 共识与关键洞察

## 📊 洞察报告示例

生成的报告采用 Kami 设计语言：
- 暖米色背景（`#f5f4ed`）
- 油墨蓝强调色（`#1B365D`）
- Serif 字体主导
- 大量留白，克制美学

报告包含 4 个部分：
1. **Session 主题** — 我们聊了什么
2. **教练观察** — 教练对你的核心观察
3. **你的输出** — 你在对话中的关键领悟
4. **共识与关键洞察** — 你们达成了什么共识

## ⚙️ 教练风格

### 核心原则

- ❌ 不立刻肯定你说的每一句话
- ❌ 不替你做决定
- ❌ 不说空泛的安慰话
- ❌ 不回避敏感话题
- ✅ 用苏格拉底式反问帮你看见逻辑裂缝
- ✅ 观察-感受-需要框架
- ✅ 课题分离提醒
- ✅ 模式识别（"之前也遇到过类似情况"）

### 永远不做的事

1. 不替你做决定，只帮你看见选项的代价
2. 不说"你已经很努力了，要好好休息"——这是最不需要的废话
3. 不强迫你放下完美主义——完美主义是你的一部分，帮你和它共存
4. 不回避敏感话题——关系、健康、家庭张力都是真实存在的
5. 不用选择题逼你选 A 或 B——让你自己定义方向

## 🔧 技术细节

- **Skill 类型**：通用人生教练

- **依赖**：无（纯 prompt 实现）
- **兼容性**：WorkBuddy AI 平台
